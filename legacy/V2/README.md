# Dante Config Editor V2

Ce dossier contient une copie de la V2 historique de Dante Config Editor.

## Important

- Cette version est fournie comme archive de référence.
- Les sources originales de la V2 ne sont pas disponibles dans ce dépôt.
- Le dossier contient uniquement les binaires compilés existants.
- La V3 a été créée séparément pour éviter de modifier directement cette version.
